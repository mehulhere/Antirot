The new app

A coach room, not another dashboard

The phone experience is built around one dominant action, voice-first check-ins, and a coach that keeps state transitions honest.

01

Voice-first coach

Speak naturally, let the backend transcribe, then get a direct coaching response without leaving the app.

02

One correct action

The home screen changes by runtime state: start, done, break, awake, good night, or the action you can actually justify.

03

Alarms that follow through

Normal and loud alarms sync with the backend. Acknowledgements, snoozes, and escalation are part of the accountability loop.

04

Current-task widget

The iOS widget keeps the current task visible so the coach is not trapped inside a chat thread.

05

Google sign-in

Native sign-in connects the phone to the managed Antirot backend without asking users to paste tokens.

06

Behavioral memory

Antirot remembers drift patterns, task history, wake behavior, and what actually gets you moving.

How it works

Talk. Act. Report. Repeat.

Buttons and voice use the same backend path. The coach can accept, challenge, or reshape your intent instead of blindly toggling state.
